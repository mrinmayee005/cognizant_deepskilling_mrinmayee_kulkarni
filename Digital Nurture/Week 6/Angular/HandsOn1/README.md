# Student Course Portal - Hands-On 1

Hands-On 1: Environment Setup, Project Structure & First Component. Covers Angular CLI, project scaffold, file explanations, creating components (header, home, course-list, student-profile). Run: ng serve

## Project Structure
- src/app/components/ - Reusable UI components
- src/app/pages/ - Page-level components
- src/app/services/ - Business logic services
- src/app/models/ - TypeScript interfaces
- src/app/guards/ - Route guards (HandsOn7+)
- src/app/interceptors/ - HTTP interceptors (HandsOn8+)
- src/app/store/ - NgRx state management (HandsOn9+)
