3. Collapse

Click the buttons and links to expand/collapse content.

Toggle Link
This content is hidden by default and can be toggled using the collapse plugin. It works with both links and buttons. The data-bs-toggle="collapse" attribute and data-bs-target (or href) attribute connect the trigger to the collapsible element.
