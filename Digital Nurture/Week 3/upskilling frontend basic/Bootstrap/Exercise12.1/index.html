Exercise 12.1 - Media Object Layout

Media Object Layout (Flex-based)

Bootstrap 5 uses d-flex instead of the old .media class. Below are examples of media object patterns.

John Doe

This is a classic media object layout. An image is placed to the left of the content using d-flex. The image is aligned to the top by default.

Jane Smith

This media object is vertically centered using align-items-center. The image and content are aligned to the middle.

Right-Aligned Image

In this layout, the content comes first and the image is placed on the right. Useful for alternating content layouts.
