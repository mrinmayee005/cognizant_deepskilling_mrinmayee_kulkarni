Fixed Footer

Scroll down! The footer bar at the bottom stays fixed to the viewport using position-fixed bottom-0.

This page demonstrates a fixed footer using Bootstrap's position utilities.

How It Works

The footer uses these Bootstrap classes:

  • position-fixed — Fixes the element relative to the viewport
  • bottom-0 — Positions it at the bottom
  • start-0 — Aligns it to the left edge
  • w-100 — Makes it full width
  • z-index-1 — Ensures it stays above other content
<footer class="position-fixed bottom-0 start-0 w-100 bg-dark text-white py-2 z-1">
    ...
</footer>

Important Notes

When using a fixed footer, add padding to the body to prevent content from being hidden behind it:

body { padding-bottom: 70px; }

Without the body padding, the last content on the page would be hidden behind the fixed footer.

Position Classes
position-fixedFixed to viewport
position-absoluteRelative to parent
position-relativeNormal flow position
position-stickySticks on scroll
bottom-0Bottom edge
top-0Top edge
start-0Left edge
end-0Right edge
