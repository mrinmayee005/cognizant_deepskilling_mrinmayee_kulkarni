Dark Gradient Section

The bg-gradient class adds a subtle gradient overlay to any contextual background color. It creates a smooth transition from a slightly lighter shade at the top to the full color at the bottom.

Custom Gradient

You can also combine bg-gradient with custom CSS gradients for unique effects.
