Centering Content with Flex Utilities

1. Horizontal Center (justify-content-center)

Centered Horizontally

This card is centered using d-flex and justify-content-center on the parent row.

<div class="d-flex justify-content-center">...</div>

2. Vertical Center (align-items-center)

Centered Vertically

Vertically centered within the parent container.

<div class="d-flex align-items-center" style="height: 250px;">...</div>

3. Perfect Center (Both Horizontal & Vertical)

Perfectly Centered

Centered both horizontally and vertically using both flex utilities together.

<div class="d-flex justify-content-center align-items-center" style="height: 300px;">...</div>

4. Text Center with Flex

Welcome to Our Site

This entire block is centered using flex column with align-items-center.

<div class="d-flex flex-column align-items-center text-center">...</div>

Flex Utilities Quick Reference
ClassPurpose
d-flexEnables flexbox on the element
justify-content-centerCenters content horizontally
justify-content-betweenSpaces content evenly with space between
justify-content-endAligns content to the right
align-items-centerCenters content vertically
align-items-startAligns content to the top
align-items-endAligns content to the bottom
flex-columnStacks flex items vertically
