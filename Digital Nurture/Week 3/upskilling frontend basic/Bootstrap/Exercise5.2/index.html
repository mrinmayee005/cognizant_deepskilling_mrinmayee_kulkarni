Exercise 5.2 - Column Reordering

Column Reordering on Different Screens

Resize the browser to see columns reorder based on screen size. On mobile, content appears first. On desktop, sidebar appears first.

Example 1: Sidebar Reorder

On mobile: Content first, Sidebar second. On desktop (md+): Sidebar first, Content second.

Sidebar (order-md-1)

This is the sidebar. On desktop it appears first (left). On mobile it appears second (bottom).

  • Dashboard
  • Settings
  • Profile
Main Content (order-md-2)

This is the main content area. On mobile it appears first (top). On desktop it appears second (right).

This pattern is useful for mobile-first designs where content should be prioritized over navigation.

<div class="col-md-4 order-md-1 order-2">Sidebar</div>
<div class="col-md-8 order-md-2 order-1">Content</div>

Example 2: Three Column Reorder

On mobile: A, B, C. On desktop (md+): C, A, B.

Column A
order-md-2
Column B

Example 3: Image/Text Swap

On mobile: Text first, Image second. On desktop: Image first, Text second.

Text Content

This text appears first on mobile for better readability. On desktop, the image takes the left position for a more visual impact.

ClassEffect
