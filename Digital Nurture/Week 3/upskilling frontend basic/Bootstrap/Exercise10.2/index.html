Responsive Navbar with Search

This exercise demonstrates a responsive Bootstrap navbar with logo, navigation links, dropdown menu, search form, and user account dropdown.

Features of this Navbar:
  • Logo: Uses navbar-brand with icon
  • Navigation Links: Standard nav items with icons
  • Dropdown Menu: "More" dropdown with additional links
  • Search Form: Input group with search button
  • User Menu: Avatar + dropdown for account settings
  • Responsive: Collapses to hamburger on mobile
Responsive Behavior

Resize your browser to see the navbar adapt:

  • Mobile: Hamburger menu, stacked layout
  • Tablet: Expandable menu
  • Desktop: Full horizontal layout
navbar-expand-lg

Controls at which breakpoint the navbar expands. lg means it stays collapsed until 992px.

data-bs-toggle="collapse"

The toggler button uses this attribute to show/hide the navbar content on mobile.

ms-auto

This margin-start auto utility pushes the search form and user menu to the right side.
