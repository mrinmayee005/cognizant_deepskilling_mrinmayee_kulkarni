Responsive Column Layout

Resize your browser to see the columns respond: Mobile: 1 column per row | Tablet (md): 2 columns per row | Desktop (lg): 3 columns per row

Column 1
col-12 col-md-6 col-lg-4

This column takes full width on mobile, half on tablets, and one-third on desktops. This demonstrates responsive grid breakpoints.

Column 2
col-12 col-md-6 col-lg-4

Same responsive behavior. On mobile this card stacks below Column 1. On tablets it sits beside Column 1. On desktops all three share a row.

Action

col-12 col-md-6 col-lg-4

The third column wraps to a new row on tablets (since only 2 fit) but aligns nicely on desktops where three columns fit per row.
