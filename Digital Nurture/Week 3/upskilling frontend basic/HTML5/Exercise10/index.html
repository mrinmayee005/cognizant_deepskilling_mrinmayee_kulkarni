Debugging with Chrome DevTools

Welcome to the Debugging Lab

This page contains intentional bugs and debugging scenarios to practice using Chrome DevTools. Open your browser's Developer Tools (F12) and follow along!

Getting Started with DevTools

  1. Press F12 or Ctrl+Shift+I (Windows) / Cmd+Option+I (Mac) to open DevTools
  2. Go to the Console tab to see log messages
  3. Go to the Elements tab and right-click elements to "Inspect" them
  4. Click the Sources tab to set breakpoints in scripts
  5. Click buttons below and watch the Console output

How to Set a Breakpoint

  1. Open DevTools and go to the Sources tab
  2. In the left panel, find <script> or look for "Snippets"
  3. Click on a line number to set a breakpoint (blue marker appears)
  4. Click the button below - execution will pause at the breakpoint
  5. Use the controls: Resume (F8), Step Over (F10), Step Into (F11)
  6. Hover over variables in the Scope panel to see their values
