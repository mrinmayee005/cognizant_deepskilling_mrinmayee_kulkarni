ID vs Class Demonstration

IDs (like #welcomeBanner): Used for UNIQUE elements. Each ID should appear only once per page.

Classes (like .highlight): Used for MULTIPLE elements. A class can be applied to many elements to share the same styling.

Block elements (div, h1-h6, p, section): Take up full width, start on a new line.

Inline elements (span, strong, em, a): Flow within text, do not start on a new line.
